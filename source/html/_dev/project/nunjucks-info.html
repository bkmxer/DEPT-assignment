This project uses Nunjucks templating engine. Visit the documentation for more info.
To enable syntax highlighting in WebStorm or PHPStorm, install the Twig plugin. Since Nunjucks using the same syntax this works perfectly.

There are 2 custom tags that have been added to Nunjucks: 'debug' and 'svg'.

svg {name}

Will inject the optimized svg contents in this place. SVG's can be added in the assets/svg folder. For an overview of which SVG's are available and their corresponding names: visit the styleguide.

{% raw %} {% svg 'social/facebook' %} {% endraw %}

Example use of the svg tag

debug {property}

Will output the property data in a debug block. Useful to see what data is available for you to use in the template. If a property is omitted it will print out the current context.

{% raw %} {% debug %}
{# -- or -- #}
 {% debug 'project' %} {% endraw %}

Example use of the debug tag
